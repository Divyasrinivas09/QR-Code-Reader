# QR Code Reader

A simple Python project that reads and decodes QR codes from image files using OpenCV.

## Features

- Detects QR codes from images
- Decodes the embedded text or URL
- Easy to use and beginner-friendly

## Technologies Used

- Python
- OpenCV

## Installation

```bash
pip install opencv-python
```

## Usage

1. Place the QR code image in the project folder.
2. Update the image filename if needed.
3. Run the script:

```bash
python qr_reader.py
```

The decoded text or URL will be displayed in the console.

## Example

**Input:**

```
QR Code.png
```

**Output:**

```
https://youtu.be/fHBR1j1kJ1I?si=VEHcM2zAwdHNIF3_
```
